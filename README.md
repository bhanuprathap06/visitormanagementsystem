##  Run the Application

###  Terminal 1 — Backend (runs migrations on boot)

```bash
cd ~/Desktop/DBMS/backend
node server.js
```

👉 http://localhost:3001

---

###  Terminal 2 — Admin Dashboard

```bash
cd ~/Desktop/DBMS/frontend
npm run dev
```

👉 http://localhost:5173

---

###  Terminal 3 — Public Site

```bash
cd ~/Desktop/DBMS/public-site
npm install
npm run dev
```

👉 http://localhost:5174
